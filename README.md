# CatalogFilter
[ MODX Revo > PDOTols > Resource filter ]

Вызов
```
[[!catalogFilter?
  &tpl=`tpl`
  &limit=`8`
  &parents=`[[*id]]`
  &fields=`price`
  &sortby=`menuindex`
  &sortdir=`ASC`
]]
```
Где _fields_ необходимые tv поля
